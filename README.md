# Sorting Algorithm Assignment

This program sorts a list of integers without using built-in sorting functions.

Language Used: C++

Algorithm Used: Bubble Sort

## Program Description
The program asks the user to enter a number of integers and then sorts them in ascending order using the Bubble Sort algorithm.

## Steps
1. The user enters the number of integers.
2. The user inputs the integers.
3. The program compares adjacent numbers.
4. Larger numbers are swapped.
5. The process repeats until the list is sorted.
