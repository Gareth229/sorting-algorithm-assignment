# Sorting Algorithm Assignment

## Algorithm
CUBE SORT (Descending)

## Language
C++

## Description
This program sorts a list of integers in descending order based on their cube values without using built-in sorting functions.

## How It Works
1. The program asks the user to enter the number of integers.
2. The user inputs the integers.
3. The cube of each number is compared.
4. Numbers are arranged in descending order according to their cube values.

## Example

Input:
5
2 4 1 3 5

Output:
5 4 3 2 1

## Time Complexity
O(n²)
